## Project Installation
Follow the steps properly -

- Flutter SDK Must be installed properly.

- Clone Repository
```bash
  gh repo clone vaibhavkusalkar/furniture_store
```

- Get into the Project folder (i.e furniture_store)
```bash
  cd furniture_store
```

- Run
```bash
  flutter clean
```
```bash
  flutter pub get
```

The App is ready to use !
